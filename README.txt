Modified version of flaskr, for devops training

Flaskr is a minimal blog application distributed as part of the Flask
Python web framework. This is a slightly modified version of it, used
in MigrateUp devops trainings.

See README.upstream and LICENSE.upstream for original
documentation. Taken from the flask 0.10.1 source code.
